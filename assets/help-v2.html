Global Navigation

Airports, Change Notices, and Manuals

Navigate to airports, view change notices, and manuals, including company content as part of your electronic subscription.

Favorite Chart, Brightness Control, Settings

Navigate through your favorite charts, application settings, and use the brightness control for changing lighting conditions.

GPS Status

Tap the GPS status icon to view parameters that the app has received from your GPS unit. This icon is available when you select it in the settings.

Airports

Airports and Favorites

Tap the All Airports icon to access and search all of the airports included in your electronic subscription. Favorites help to make airports you use frequently easier to find.

Favorite an Airport

Tap on the empty star icon to add an airport to the favorites view. Tap on the filled star icon to remove favorite status.

Charts

All Charts

Tap the All Charts icon to view all of the charts including favorites and groups.

Charts and Favorites

Tap the double star icon to view your favorite charts. Groups view allows you to view the charts sorted by groups.

Groups

Charts are grouped in two ways: all groups, or sorted based on chart type (SID, STAR, etc).

Favorite a Chart: Popover

Tap on the empty star icon to add a chart to the favorites view. Tap on the filled star icon to remove favorites status.

Favorite a Chart: Title Bar

When viewing a chart, tap the empty star icon to make it a favorite. Tap the double star icon to view all favorite charts.

Chart Change Notices

While viewing a chart tap the chart change notices icon to view available change notices.

Chart Selection Scrubber

To see the longer name of a chart that is not currently being viewed, use a long press on that chart index number. The tool tip that opens includes a favorites star if that chart is set as a favorite.

Manuals

Viewing Manuals

View the manuals and company manuals you have loaded from the Manuals icon. Loaded manuals will be listed and will update with other data updates. Tap the one you want to view.

Downloading and Updating Manuals

To update or download manuals, select or deselect the manuals in your subscription, tap Settings > Updates > Manuals and select or clear the check box for each available manuals.

Brightness Control

Application Brightness

Tap the starburst icon to open the dimmer slider and adjust the screen's brightness. This control only changes the brightness of the app, not the device.

Settings

Updates

Tap Settings > Updates to check the chart cycle and the next chart issue date, the last manuals update, and for any available updates.

Preferences

Tap Settings > Preferences to restore quick tips, display your ownship, turn off sleep mode, enable company content, and login if necessary.

Display Ownship—Select to display an ownship icon superimposed on airport diagrams. You also need to select Device Settings > Location and security and select Use GPS satellites.

Help

Tap Settings > Help to view this help.

About

Tap Settings > About to view the site key, the version, to deactivate the app, and to view the application end user licence agreement and the data end user license agreement.

To deactivate, release the site key, and remove all data, tap the Deactivate icon and follow the on-screen instructions.
